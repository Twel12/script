# PixelOS build_scripts for davinci

## Usage:
```bash
# create a folder to hold source code
mkdir PixelOS
cd PixelOS

# clone build_scripts into aosp
git clone https://github.com/PixelOS-and-Not-So-Pixel/script.git script

# run build scripts
bash script/server.sh
```
